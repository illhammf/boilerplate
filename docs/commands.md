# ⚡ Commands

## dcu

```bash
dcu
```

Start Docker.

---

## dcd

```bash
dcd
```

Stop Docker.

---

## dcm

```bash
dcm User
```

Generate:

- Model
- Migration
- Seeder
- Controller
- Filament Resource

---

## dcr

Remove generated files.

---

## dcp

```bash
dcp "update user model"
```

Automatically:

- git add
- git commit
- git pull --rebase
- git push

---

## dci

Initialize project.
